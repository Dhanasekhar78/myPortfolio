## ✨ Features of this Portfolio

* **Dynamic Years of Experience (YOE):** Embedded JavaScript calculates fractional experience since November 2023 on page load.
* **Modern Aesthetic:** Slate-charcoal background with indigo accent gradients, matching dashboard aesthetics used by modern tech platforms (Linear, Vercel).
* **100% Responsive Grid Layout:** Seamless fluid typography and elements optimized for mobile, tablet, and 4K displays.
* **Vector Icons:** Scalable inline SVG indicators that transition dynamically on user hover actions.
* **Direct Connect Cards:** Direct integration for LinkedIn messaging and professional email contact.

## 🚀 Local Setup

To view and edit the portfolio locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Dhanasekhar78/myPortfolio.git
   ```
2. Navigate to the folder:
   ```bash
   cd myPortfolio
   ```
3. Open `index.html` in any web browser to run the static site locally.
